The Module Extender For Divi adds the capacity to import custom modules into the Divi Page Builder. Build your own or choose from our modules list.

== Description ==
The Module Extender For Divi works with the Divi Page Builder by Elegant Themes. Simply add module zip files to the child theme’s modules folder. Then the additional modules will be available in the Divi Page Builder “Add Module” list.


As modules become available you can choose those you want in download them. Or you can create your own modules and upload them into your child theme. This platform makes extending Divi Page Builder easy.


== Installation ==
1. Download the zip file
2. Install using WordPress plugin add new
3. start adding modules.
